Windows Calculator

by Gustavo Campos

Welcome to W10 Calculator!

This calculator was created by Gustavo Campos, a new one software engineer.

Both the design, and display logic of this calculator are intended to look similar as possible to a calculator that you can find in every computer with Windows 10 installed.

As the original, you have either the option to click on the buttons than the option to use your keyboard to interact.

Commands

  • Numbers keys: Do the same thing that the number buttons.
  • + key: Sum
  • - key: Subtract
  • * key: Multiplicate
  • / key: Divide
  • = key: Get the result
  • . key: Add a point on display number
  • Shift: Invert the display number
  • Ctrl: Clean display number
  • Back Space: Exactly what a backspace is supposed to do
  • C key: Clear everything
